Box-car Filter¶

Description¶

A basic 1-D boxcar (moving average) filter

Signature¶

filter/boxcar_filter_1d

Required Parameters¶

in1¶

Variable from which to read input values

out1¶

Variable in which to store output values

Number of samples to store in memory and average over
