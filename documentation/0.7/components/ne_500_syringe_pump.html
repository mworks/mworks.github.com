NE-500 Syringe Pump¶

Description¶

Used for interfacing to a specific pump within a NE-500 Syringe Pump Network

Signature¶

iochannel/ne500

Required Parameters¶

capability¶

Example:00

Network address of pump (0 to 99)

syringe_diameter¶

Example:15.0

Syringe diameter (millimeters)

flow_rate¶

Example:5

Pumping rate (units given by rate_unit)

variable¶

Variable from which to read the volume to dispense. Assigning a nonzero value to the variable causes MWorks to prepare the pump’s onboard dispensing program (updating rate, volume, and direction as needed). If the pump network is enabled, pumping will then begin.

Optional Parameters¶

rate_unit¶

Options:UM, MM, UH, MH
Default:MM

Units of flow_rate. The options are uL/min, mL/min, uL/hr, and mL/hr.

volume_unit¶

Options:UL, ML

Volume units.

If omitted, volume units are determined by syringe_diameter. If the syringe diameter is less than or equal to 14mm, UL is used. Otherwise, ML is used.
