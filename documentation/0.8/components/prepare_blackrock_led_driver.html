Prepare Blackrock LED Driver¶

Available on macOS

Description¶

Prepare a Blackrock LED Driver to run for the specified duration. The desired channel intensities must have been set previously.

Signature¶

action/blackrock_led_driver_prepare

Required Parameters¶

device¶

Device name

duration¶

Run duration (microseconds)
